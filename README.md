# Personal Finance Analytics System

## Overview
An end-to-end analytics system designed to aggregate multi-source financial data and support budgeting, forecasting, and investment decision-making.

## Business Problem
Personal financial data is fragmented across multiple accounts and platforms, making it difficult to form a unified view of spending, cash flow, and financial health. Manual reconciliation is time-consuming and error-prone.

## Data
- Multi-source bank statements
- Transaction-level expense data
- Historical financial records

## Analytical Approach
- Designed a unified data model to standardize transactions across sources
- Built analytical workflows for spending analysis and cash flow forecasting
- Developed decision-support outputs to evaluate budgeting and optimization scenarios

## Key Outputs
- Consolidated financial summaries
- Forecasts of monthly cash flow
- Actionable insights for budgeting and allocation decisions
